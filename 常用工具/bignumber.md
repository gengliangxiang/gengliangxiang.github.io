# Bignumber

## 引入

```
import BigNumber from 'bignumber.js'
```

## 加法 `plus`

```
let x = new BigNumber(0.1);
x.plus(0.3)
```

## 减法 `minus`

```
let x = new BigNumber(0.3);
x.minus(0.1)
```

## 乘法 `multipliedBy`

```
let x = new BigNumber(0.3);
x.times(0.1)
// or
x.multipliedBy(0.1)
```

## 除法 `dividedBy`

```
let x = new BigNumber(0.3);
x.dividedBy(0.1)
// or
x.div(0.1)
```
